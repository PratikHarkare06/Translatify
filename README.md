# Translatify üåçüó£Ô∏è

Translatify is an immersive language learning application that pairs you with an intelligent AI tutor, "Sofia," for real-time voice conversations. Powered by Google's Gemini 2.5 Native Audio (Live API), it provides a seamless, low-latency conversational experience to help you master new languages.

## ‚ú® Features

- **Real-time Voice Practice**: Have natural, spoken conversations with Sofia in multiple languages. The AI responds instantly with human-like intonation.
- **Live Translation**: Stuck on a phrase during practice? Speak in your native language, and the app will translate it for you within the conversation context.
- **Smart Feedback**: Request detailed analysis of your practice sessions. Get fluency scores, grammar corrections, and phrasing suggestions generated by Gemini.
- **Session History**: Review transcripts of your past conversations to track your progress.
- **Universal Translator**: A dedicated tool for text-to-text and speech-to-text translation with text-to-speech playback.
- **Customizable Voices**: Choose from different AI voice personalities (Zephyr, Puck, Charon, Kore, Fenrir).

## üõ†Ô∏è Tech Stack

- **Frontend**: React 19, TypeScript
- **Styling**: Tailwind CSS
- **AI Integration**: Google GenAI SDK (`@google/genai`)
  - **Live API**: `gemini-2.5-flash-native-audio-preview-09-2025` for real-time audio streaming.
  - **Text Generation**: `gemini-2.5-flash` for translations and feedback.
  - **TTS**: `gemini-2.5-flash-preview-tts` for high-quality speech synthesis.
- **Audio Processing**: Web Audio API (AudioContext, ScriptProcessorNode) for raw PCM audio streaming and encoding/decoding.

## üöÄ Getting Started

### Prerequisites

- Node.js installed on your machine.
- A Google Gemini API Key. You can obtain one from [Google AI Studio](https://aistudio.google.com/).

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/translatify.git
   cd translatify
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure API Key**
   The application requires the `API_KEY` environment variable to function.
   
   Ensure your environment or build tool is configured to expose `process.env.API_KEY`. For example, you might use a `.env` file or system variables depending on your build setup.
   
   ```env
   API_KEY=your_actual_api_key_here
   ```

4. **Run the application**
   ```bash
   npm start
   ```

## üìñ Usage

### Practice Mode
1. Select your **Native Language** and the **Language you want to learn**.
2. Choose an AI voice for Sofia.
3. Click **"Start Practicing"**.
4. Speak into your microphone. Sofia will reply in the target language.
5. Use the **"Translate Phrase"** button if you need to say something in your native language and have it translated into the conversation.

### History & Feedback
1. Navigate to the **History** tab.
2. Click on a past session to view the transcript.
3. Click **"Get Feedback from Gemini"** to generate a report on your performance.

### Translator
1. Navigate to the **Translate** tab.
2. Type text or use the **Microphone** icon to speak.
3. Click **Translate** to see the result.
4. Click the **Speaker** icon to hear the translation pronounced.

## üîí Privacy & Permissions

- **Microphone Access**: The app requires microphone permission to stream audio to the Gemini Live API.
- **Data**: Session transcripts are stored locally in your browser's `localStorage` for your history view.
